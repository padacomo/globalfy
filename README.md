# Project

POC - globalfy events calendar

## 🚀 starting

Follow the instructions below to run the code.

### 📋 requirements

🛠️ Nest JS 🛠️ docker-compose 🛠️ Insomnia or Postman

### 📌 As this is a POC, the .env file is not ignored.

### 🔧 Installation

Install dependencies

```
npm install
```

run docker compose

```
docker-compose up -d
```

run the project

```
npm start
```
